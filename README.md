# 🎭 JokeAPI Web App

A simple and fun web app that fetches random jokes using an external API and displays them beautifully in the browser. Perfect for learning **Express**, **Axios**, and **EJS templating**.

---

## ✨ Features

* 🎲 Fetches random jokes from an API
* 🧠 Supports both:

  * **Single jokes**
  * **Two-part jokes (setup + delivery)**
* 🎨 Styled with custom CSS
* ⚡ Fast and lightweight Express server

---

## 🛠️ Tech Stack

* **Node.js**
* **Express**
* **Axios**
* **EJS**
* **CSS**

---

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
npm i
```

### 3. Run the app

```bash
nodemon index.js
```

---

## 🌐 Open in Browser

Once running, go to:

```
http://localhost:3000
```
